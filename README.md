## 如何使用

### 典型工作流示例

**场景 1：从零写一篇公众号长文**
```
/zhicai-全流程编排 → 自动串联选题→标题→开头→结构→段落→风格→情感→结尾
                    → 完成后用 /zhicai-爆款预测 评分
                    → 用 /zhicai-AI去味 做最终润色
```

**场景 2：遇到写作瓶颈**
```
/zhicai-内容诊断 → 定位根因层（心理/技法/结构/情感/系统）
                  → 匹配对应创作者工具包
                  → 获得 3 步行动计划
```

**场景 3：把一篇公众号文章分发到小红书**
```
/zhicai-跨平台分发 → 字数压缩/结构重组/风格调校
                    → 输出适配后版本
```

## 如何安装

### Claude Code（推荐）

```bash
claude plugin marketplace add hanzhicaicode/zhicai
claude plugin install zhicai@zhicai-skills
```

### 通用方式（Codex / Claude Code）

```bash
npx -y skills add hanzhicaicode/zhicai -g --all
```

### 本地构建

```bash
bash tools/build-skills.sh
```

产物在 `dist/skills/zhicai-{VERSION}.zip`。

### 如何更新

#### Claude Code
```bash
claude plugin marketplace update zhicai-skills
claude plugin update zhicai@zhicai-skills
/reload-plugins
```

#### 通用
```bash
npx -y skills add hanzhicaicode/zhicai -g --all
```
