## 如何使用

### 典型工作流示例

**场景 1：从零写一篇公众号长文**
```
/zhicai-全流程编排 → 自动串联选题→标题→开头→结构→段落→风格→情感→结尾
                    → 完成后用 /zhicai-爆款预测 评分
                    → 用 /zhicai-AI去味 做最终润色
```

**场景 2：遇到写作瓶颈**
```
/zhicai-内容诊断 → 定位根因层（心理/技法/结构/情感/系统）
                  → 匹配对应创作者工具包
                  → 获得 3 步行动计划
```

**场景 3：把一篇公众号文章分发到小红书**
```
/zhicai-跨平台分发 → 字数压缩/结构重组/风格调校
                    → 输出适配后版本
```

## 如何安装

### Claude Code（推荐）

```bash
claude plugin marketplace add rszkck/zhicai-main
claude plugin install zhicai@zhicai-skills
```

### 通用方式（所有支持 npx skills 的 agent）

包括 Claude Code、Cursor、Cline、Windsurf 等 55+ agent：

```bash
npx -y skills add rszkck/zhicai-main -g --all
```

### Hermes

```bash
# 逐个部署
for skill in skills/*/; do
  name=$(basename "$skill")
  mkdir -p ~/.hermes/skills/"$name"
  cp "$skill/SKILL.md" ~/.hermes/skills/"$name"/SKILL.md
done
```

或使用仓库中的一键脚本：

```bash
bash scripts/deploy-hermes.sh
```

### Trae Solo

从 [GitHub Releases](https://github.com/rszkck/zhicai-main/releases) 下载 `zhicai-1.0.0.zip`，解压后将 19 个 skill zip 逐个拖入 Trae Solo 的上传技能窗口。

### 本地构建

```bash
bash tools/build-skills.sh
```

产物在 `dist/skills/zhicai-{VERSION}.zip`。

### 如何更新

#### Claude Code
```bash
claude plugin marketplace update zhicai-skills
claude plugin update zhicai@zhicai-skills
/reload-plugins
```

#### 通用 / Hermes
重新运行安装命令即可。
